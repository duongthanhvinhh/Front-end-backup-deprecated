There are 3 types of list: Unordered list, ordered list and Other lists

Ordered List

An ordered list starts with the ol tag. Each list item starts with the li tag.
The list item will be marked with the numbers by default:

  1. Line 1
  2. Line 2
  3. Line 3
  4. Line 4
  5. Line 5

We have some attribute types in the ordered HTML list like '1', 'A', 'a', 'I', 'i'

  1. vinh
  2. vinh1
  3. vinh2
  4. vinh3

Also We can contrl list counting

  1. first
  2. second
  3. third
  4. fourth

Nested list, same as Unordered list. Note that a list item (li) can contain a new list, and other HTML elements, like images and links, etc.
