This is Block and Inline Elements section

Block-level element always starts on a new line, and the browsers automatically add some space(a margin) before and after the element

        Here are the block-level elements in HTML:
        address,article,aside,blockquote,canvas,dd,div,dl,dt,fieldset,figcaption,figure,footer,form,h1->h6,header,hr,li,main,nav,noscript,ol,p,pre,section,table,tfoot,ul,video

p is block element while span is inline element.

        Here are the inline-level elements in HTML:
        a,abbr,acronym,b,bdo,big,br,button,cite,code,dfn,em,i,img,imput,kbd,label,map,object,output,q,samp,script,select,small,span,strong,sub,sup,textarea,time,tt,var
